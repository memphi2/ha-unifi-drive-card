# Drive Storage Card 0.2.3

Patch release focused on rendering correctness, discovery filtering safety and
release hardening.

## Highlights

- Fixed cached section rendering so busy action indicators update immediately
  during in-flight service calls.
- Adjusted discovery filtering to keep integration entities without registry
  `device_id` when a card device is selected.
- Added locale/formatter dependencies to section cache signatures so language
  and formatting changes re-render without waiting for state object churn.
- Refined end-user docs with a faster setup flow, practical configuration
  recipes and clearer troubleshooting steps.

## Validation

- TypeScript typecheck, ESLint, Vitest and Vite build.
- HACS/custom-card compatibility checks.
- License and third-party notice checks.
- Anonymization scan.
- Browser render smoke.
- Live Home Assistant install/uninstall smoke.
- Moderate security audit.
- Release metadata and workflow checks.
