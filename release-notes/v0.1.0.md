# Drive Storage Card 0.1.0

Initial public release for the Home Assistant `unifi_drive` Lovelace card.

## Highlights

- Compact dashboard card for compatible UniFi Drive / UNAS entities.
- Automatic discovery from Home Assistant entity registry metadata.
- Dynamic sections for storage, pools, drives, snapshots, system, updates,
  diagnostics and backup actions.
- Native controls for switches, buttons, numbers, selects, time entities and
  update entities.
- Width-aware layout that reorders blocks between vertical and wide horizontal
  views when the dashboard card width changes.
- Safe defaults: restart and shutdown actions stay hidden unless explicitly
  enabled and still require confirmation.
- Visual editor for sections, compact layout, diagnostics, entity overrides,
  hidden entities and Home Assistant actions.
- Home Assistant `perform-action` support for service actions, target fields and
  JSON service data.

## Validation

- TypeScript typecheck, ESLint, Vitest and Vite build.
- HACS/custom-card compatibility checks.
- License notice validation for bundled Lit runtime packages.
- Anonymization scan for committed text files.
- Browser render smoke with a regenerated screenshot.
- Live Home Assistant install and uninstall smoke.
