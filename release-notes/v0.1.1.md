# Drive Storage Card 0.1.1

Maintenance and refactor release focused on editor quality and release hygiene.

## Highlights

- Refactored the card editor into reusable shared modules for text/checkbox/picker
  value parsing and action-state handling.
- Standardized section and overview ordering controls with shared button rendering
  while keeping drag/drop and keyboard interactions unchanged.
- Expanded editor tests with shared setup helpers and cleaner selector interaction
  coverage.
- Improved release metadata checks to be version-aware for future releases while
  preserving existing compliance and trademark checks.

## Validation

- TypeScript typecheck, ESLint, Vitest, Vite build.
- HACS/custom-card compatibility checks.
- License and third-party notice checks.
- Anonymization scan.
- Release metadata and workflow checks.
- Moderate security audit.
