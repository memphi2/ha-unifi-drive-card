# Drive Storage Card 0.3.3

Patch release focused on typography and value readability polish for overview
and regular entity rows.

## Highlights

- Stabilized overview metric text rendering by using explicit metric label/value
  classes to keep compact tiles readable at narrow widths.
- Matched regular entity-row label typography to Home Assistant body-text
  tokens for native-looking labels.
- Increased non-overview value prominence slightly to improve scanning without
  changing existing interaction controls.

## Validation

- TypeScript typecheck, ESLint, Vitest and Vite build.
- HACS/custom-card compatibility checks.
- License and third-party notice checks.
- Local anonymization scan and GitHub-surface anonymization scan.
- Live Home Assistant deploy smoke and live entity audit.
- Release metadata and workflow checks.
