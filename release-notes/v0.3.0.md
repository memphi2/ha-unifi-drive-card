# Drive Storage Card 0.3.0

Minor release focused on DHE-parity alignment, modular refactoring and runtime
performance hardening.

## Highlights

- Aligned the editor UX/behavior with the latest `ha-dhe-connect-card` patterns
  while preserving Home Assistant-native configuration flow.
- Refactored section/entity editor rendering into clearer modular building
  blocks with less duplication.
- Improved runtime/editor hot paths with cache-based optimizations:
  - hidden entity lookup set caching
  - entity override key caching
  - discovery config signature caching
  - reduced section grouping render work
- Hardened shared editor event parsing for `value`, `device_id` and `deviceId`
  event payload variants.
- Added focused tests for shared editor helpers and parity-critical editor
  behavior.
