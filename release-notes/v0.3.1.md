# Drive Storage Card 0.3.1

Patch release focused on editor/runtime consistency and HA deploy smoke
robustness on SMB-mounted config paths.

## Highlights

- Fixed dangerous-entity checkbox behavior in the visual editor so
  `show_dangerous_actions` is auto-enabled when enabling dangerous actions and
  dangerous keys stay hidden when that switch is disabled.
- Added targeted editor tests for dangerous-action visibility and update flow.
- Improved HA install/deploy smoke copy behavior on GVFS/CIFS paths by adding a
  copy fallback for filesystems that reject native `copyFile`.

## Validation

- TypeScript typecheck, ESLint, Vitest and Vite build.
- HACS/custom-card compatibility checks.
- License and third-party notice checks.
- Anonymization scan.
- Browser render smoke.
- Live Home Assistant install/deploy and install/uninstall smoke.
- Release metadata and workflow checks.
